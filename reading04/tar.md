TLDR - tar
==========

Overview
--------

**tar** - archiving utility for manipulating .tar files

Examples
--------

- Extract a file's contents
	
	$ tar -zxvf file.tar.gz

- Create an archive

	$ tar cvzf data.tar.gz {location}
