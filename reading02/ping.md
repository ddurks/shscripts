TLDR - ping
==========

Overview
--------

**ping** - sends an ICMP ECHO_REQUEST to network hosts
	allows the user to determine the status of the network connection

Examples
--------

- simply sends the ping to the host

	$ ping

- audible ping

	$ ping -a

- print timestamp before each line

	$ ping -D
