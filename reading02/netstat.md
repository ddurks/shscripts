TLDR - netstat
==========

Overview
--------

**netstat** - prints network connections, routing tables, interface statistics
	masquerade connections, and multicast memberships

Examples
--------

- display the netstat of a given ip address

	$ netstat <ipaddress>

- display netstat of current machine

	$ netstat
