TLDR - tr
==========

Overview
--------

**tr** - translate or delete characters

Examples
--------

- translate from lower to upper case

	$ tr [:lower:] [:upper:]

- delete specified characters from a string

	$ tr -d aeiou

- suppress extra whitespace in a string

	$ tr -s [:blank:]
